📌 Customer Churn Prediction using Machine Learning

This project predicts whether a customer will churn (leave a service) using machine learning techniques on a telecom dataset.
---
🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Random Forest Classification Model
- Model Evaluation (Accuracy, Confusion Matrix)
- Feature Importance Analysis
---
🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Google Colab  
---
📊 Workflow
1. Data Loading  
2. Data Cleaning  
3. Exploratory Data Analysis  
4. Feature Encoding  
5. Model Training  
6. Model Evaluation  
7. Insights  
---
📂 Files
- `Customer_Churn_Prediction.ipynb` → Main notebook  
---
💡 Key Insights
- Customers with higher monthly charges are more likely to churn  
- Contract type significantly impacts churn  
- Longer tenure reduces churn probability
